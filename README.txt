1.Detection项目用于拍摄图片，得到多帧8192×2048的图片，并截取每张图片中间2048×2048的图片拼接起来得到一张大图，这张大图用BatchTest进行处理得到红黑图与excel。运动时相机运行拍摄一长条图像，分成了好多帧，例如1-1.bmp，1-2.bmp，如果平台运动宽度设置的足够长，相机移动拍摄完一长条图像后会沿垂直方向移动42mm，继续拍摄第二条第三条等，例如2-1.bmp，2-2.bmp；3-1.bmp，3-2.bmp。最终1-x.bmp截取中间2048×2048纵向拼接合成的大图为1-unite.BMP，2-x.bmp截取中间2048×2048纵向拼接合成的大图为2-unite.BMP，以此类推。

2.此项目需配置opencv、MPC2810、Sapera LT三个库
MPC2810库下载：www.leetro.com/down/MPC2810A%20V1.0.17-32bit%20Win7.rar
Sapera LT库下载：sapera_lt_831_sdk.exe 或者在http://51camera.com.cn/show-25-12-1.html的“软件及例程下载”中下载，这个网址就是此项目相机的选型。注意：SapClassGuiD.lib、SapClassGui.lib以及对应的dll要用它里面给的工程重新生成一下，具体操作见文档“Dalsa VS2012编程环境下生成SapClassGui.Lib以及DLL方法.doc”

已经在项目中属性管理器的Debug|x64和Release|x64配置了这几个库的属性表，使用时请以按照自己电脑的环境修改。
2.1修改方法一：可右击属性页->属性，修改包含目录、库目录和附加依赖项这几个选项。
2.2修改方法二：将属性页中用到的这三个库的.h文件 .lib文件 .dll文件全部放到源程序文件夹。

3.关于项目的一些参数
电机：86步进电机
电机转1圈前进10mm（这个项目做了两台机器，这里指第一台机器，第一台机器与第二台电机型号不同，其他都相同）
电机接受200个脉冲前进1mm
相机接受1个脉冲取1行
所以相机前进1mm扫描200行，所以在移动方向一个像素就是5μm。

那么原图就是
宽8192像素，实际44mm
宽一个像素5.371μm（但项目一直沿用5.378）
高一个像素为5μm

应设置扫描行数为2048*5.378/5=2203，再进行压缩，压缩至2048，这样长宽比例一致。

得到的一帧图像为8k*2k
